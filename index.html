Como configurar o mailto no Windows para abrir o Gmail no Chrome

1. Definir o Chrome como navegador padrão

Abra Configurações do Windows → Aplicativos → Aplicativos padrão. Em Navegador da Web, escolha Google Chrome.

2. Ativar o Gmail como manipulador de email no Chrome

Abra o Chrome e acesse mail.google.com. Faça login na sua conta. Na barra de endereço, procure o ícone de losango com um envelope (lado direito). Clique nele e selecione Permitir. Confirme o Gmail como aplicativo padrão para links de email.
